Message Type

There are several types of FTP: FTP, FTPS and FTPES (Grant, n.d.).

The traditional FTP is unencrypted and can be used over most web browsers. Traditional FTP requires the use of an unencrypted username and password to login and transfer files over the network (BasuMallick, 2022). The sending of unencrypted authentication information over the network exposes the data to hackers gaining unauthorized access, and it is thus an unsecure method of transferring files.

FTPS, also known as File Transfer Protocol Secure, has an added transport layer security to encrypt the data being transferred. This added layer requires a secondary connection between the client and server (BasuMallick, 2022).

As with FTPS, FTPES also adds a layer of encryption, however FTPES connects to a web hosting account rather than an implicit connection. FTPES is a relatively newer version of the FTP and is the recommended option for establishing secure connections (Grant, n.d.).
