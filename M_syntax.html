Message Syntax

FTP uses telnet character strings for commands. There are three different types of FTP commands: access control commands, transfer parameter commands, FTP service commands (Postel and Reynolds, 1985). Access control commands include, but are not limited to User Name (USER), Password (PASS), Account (ACCT) and Logout (Quit). Transfer parameter commands include Data Port (PORT), Passive (PASV), Representation Type (Type), File Structure (STRU) and Transfer Mode (MODE). FTP service commands include, but are not limited to Retrieve (RETR), Store (STOR), Restart (REST) and Abort (ABOR) (Postel and Reynolds, 1985).
